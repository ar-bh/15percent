# 🪚 Magic Chainsaw
Magic Chainsaw is a fun indie game where you use your mouse (the chainsaw, yet to be added) to cut the shapes and trim their area down to 15% or less. But you have to be careful because if you slip up once, and separate the pointers or creatures inside, you lose!
Made with Godot 4.3 I think

## Gameplay!
When you open the game or the git page, you are met with the start, instructions, and quit button. The instructions tells you everything you need to know but I still wrote it below. Pressing start starts the game (obviously) and quit does the same when the game is being ran in an individual window. If you lose by seperating the pointers, you have to restart or can visit the main menu again.

##Instructions
Cut the shapes and trim the area!
But do not seperate the pointers.

You can enter cutting mode by either pressing E or holding down the left mouse. Then use the mouse and make quick cuts to win!
I added lots of restraints on what counts as a cut.
For example, you cannot be cutting too slow, you have to be in cutting mode, and you cannot cut a line that is formed by only one side of the polygon (it has to be 2 to draw a line and make the cut)
